# Simple Flashcards – Docs Site

Static website for the **Simple Flashcards** Android app, published by **Skypedia**.  
Provides a landing page, Privacy Policy, and Terms of Service for Google Play listing.

## Pages

| File                  | URL                    |
| --------------------- | ---------------------- |
| `index.html`          | `/`                    |
| `privacy-policy.html` | `/privacy-policy.html` |
| `terms.html`          | `/terms.html`          |

## Structure

```
/
├── index.html           # Landing page
├── privacy-policy.html  # Privacy policy
├── terms.html           # Terms of service
├── styles.css           # Shared stylesheet
└── README.md            # This file
```

## Stack

- Pure HTML + CSS — no frameworks, no build tools, no dependencies
- Mobile-friendly responsive layout
- Material Design inspired, light theme
- Suitable for GitHub Pages static hosting
